# FalcoEval：大模型性能自动化评估系统

随着大语言模型技术的快速发展，模型规模与能力呈现爆炸式增长，但其性能评估仍面临以下挑战：
- 评估流程碎片化：现有评估多依赖人工编写脚本、手动切换数据集与指标，流程繁琐且易出错，单次完整评估需耗费数小时至数天。
- 评估标准不统一：不同研究团队采用各异的数据集子集、预处理方式及指标计算方法，导致模型性能结果缺乏横向可比性，阻碍技术进步的客观衡量。
- 多能力维度覆盖不足：单一数据集难以全面反映模型在数学推理、常识理解、专业领域知识、多跳推理等多维度能力，需跨数十个专业数据集进行系统性评测。
- 专业领域评估缺失：针对法律、医学、数学竞赛等高专业性场景的评估工具稀缺，难以验证模型在关键垂直领域的可靠性与安全性。

## What is it?
**FalcoEval**通过构建统一评估框架，实现以下价值：
- 自动化流水线: 集成预测生成、参考答案构建、指标计算全流程，用户仅需选择模型与数据集即可一键完成评估，效率显著提升。
- 多维度评估: 支持50+国际主流基准数据集（涵盖ARC、MMLU、GSM8K、HotpotQA等），覆盖常识推理、数学能力、多跳问答、专业领域知识等12类核心能力维度。针对法律（LEXam）、医学（MedQA）、数学竞赛（AIME、MATH-500）、研究生级知识（GPQA）等高难度场景，提供领域定制化提示词模板与评估逻辑，满足专业场景的严苛验证需求。
- 标准化指标: 内置Accuracy、Exact Match (EM)、F1、ROUGE、BLEU、Token Efficiency等10+专业指标，严格遵循各数据集官方评估规范，确保结果可复现、可对比。
- 开放可扩展架构: 支持HuggingFace格式的任意开源模型（如LLaMA、Qwen、ChatGLM系列）及本地私有化部署模型，用户可便捷接入自定义数据集与评估指标。

## Quickstart
```sh
conda create -n falcoeval python=3.11 -y
conda activate falcoeval
pip install nltk numpy pandas rouge_score tdqm gradio transformers torch
python main.py
```
